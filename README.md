# JSAmazon
Show how to automatizate mobile app with JS + Mocha + Appium


# Install
Install [npm](https://nodejs.org/en/)
Go to folder and use command:
```
npm install
```

# Run
Start appium in terminal/cmd
```
appium
```

Open new tab in terminal and use command: 
```
npm run wdio
```
